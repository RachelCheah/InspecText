Telegram

How to export Telegram chat:

  1. Install Telegram Desktop and complete the installation process

  2. Open Telegram Desktop app

  3. Open chat

  4. Click More Options

  5. Click Export chat history

  6. Untick the Photos selection

  7. Change the Format from HTML to JSON

  8. Check that your settings is the same as shown below. Keep the start and end date as the default selection, you can choose to customize that later on!
  9. Click EXPORT

  10. After "Your data was successfully exported", click SHOW MY DATA. This should bring you to the export folder

  11. Upload the file result.json using the button below
