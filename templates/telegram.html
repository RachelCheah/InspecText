Telegram

How to export Telegram chat:

  1. Install the Save Telegram Chat History chrome extension
  2. Login to Telegram web
  3. Open chat
  4. Click on the extension icon on the top right hand corner of your browser
  5. Choose the All option under "Load", or any number of messages you want to include in the Inspection
  6. Click the save icon at the bottom to obtain a .txt file
